Interview Scheduler

GitHub link

Interview Scheduler is a single-page application. Front-end: React, JavaScript, HTML, CSS, SASS, AJAX. Back-end: Node JS, Express. Database: PostgreSQL. Testing: Storybook, Jest, Cypress.


As a user you can:

- add/edit/delete interviews

- choose an interviewer

- add your name

- see available spots for an interview during a week

- see the confirmation page before performing any actions

Developmet

This app was built with functional components. I started with creating a small components and tested them in isolation in Storybook. After that I added thees components to the main app component and passed props. To comunicate with back-end I cohose AXIOS library. The most interesting thing which I learned during the development is Cypress, wich I used to make end-to-end testing.
